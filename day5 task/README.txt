Day 5 – Resume Text Extraction Engine

Objective:
Build a system to extract and clean resume data for AI ATS.

Features:
- PDF & DOCX reading
- Text cleaning & normalization
- Section extraction
- JSON output
- Logging system

How to Run:
1. Install:
   pip install PyPDF2 python-docx

2. Run:
   python main.py

Output:
- JSON files in output/
- Logs in logs/
